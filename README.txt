Alexa AppKit Documentation

The documentation for the Alexa AppKit is now available on the Amazon Apps and Services Developer Portal.

To access the documentation, you need to first log in using your developer account:

1. Log in with your developer account at https://developer.amazon.com/login.html.
   If you are already logged in, the home page will load.

2. Go to https://developer.amazon.com/appsandservices/solutions/devices/echo/alexa-app-kit
   and follow the links on the page to access the detailed documentation.

3. If the documentation does not load, try refreshing the page, or log in again to renew your
   session.
